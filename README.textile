h1. junit-testdata-rule

p. JUnit TestRule to load data into a database before a test runs.

h2. Usage

bc.. 
  public class YourTestCase {
  
    @Rule
    public TestData testData = ...;
    
    @Test
    @Load("customers.sql")
    public void shouldSeeCustomersInDatabase() {
      // ...
    }
  
  }
  
h2. Integration

p. Spring & Guice have their own module. It supplies a default configuration for the TestData rule which should be suitable for the majority of use-cases.
